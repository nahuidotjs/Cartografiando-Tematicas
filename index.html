📸 Cómo usar imágenes de Google Drive en este catálogo

Para agregar una imagen alojada en Google Drive, sigue estos pasos:

  1. Sube tu imagen a Google Drive.
  2. Comparte el archivo con "Cualquier persona con el enlace" (modo visor).
  3. Copia el ID del archivo de la URL: https://drive.google.com/file/d/ID_DEL_ARCHIVO/view
  4. Usa la siguiente URL directa: https://drive.google.com/uc?export=view&id=ID_DEL_ARCHIVO
  5. Pega esa URL en el campo imageUrl dentro del array de ediciones o ganadores (ver comentarios en el código JavaScript).

Ejemplo funcional: https://drive.google.com/uc?export=view&id=1ABC123xyz

Si prefieres usar imágenes externas, cualquier URL pública (jpg, png) también funciona.
